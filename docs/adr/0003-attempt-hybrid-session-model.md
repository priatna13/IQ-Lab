# Attempt with hybrid Domain Sessions (pause between, timer within)

Full nine-Domain Assessment is long (60–90+ minutes). Forcing a single sitting destroys completion; fully untimed Domain Sessions weaken score comparability. An Attempt contains ordered Domain Sessions: Participant may pause between Domains; within a Domain Session the timer is server-authoritative and Responses are scored only on the server.

**Consequences:** Progress persistence and Account-before-start are mandatory; client timers are display-only; integrity is “light” (log blur/tab, no auto-fail).
