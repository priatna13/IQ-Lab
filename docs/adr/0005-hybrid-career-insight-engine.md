# Rule Payload is truth; LLM only narrates Insight

Career guidance must be explainable and track-specific (`explore` vs `career`). A deterministic rule engine maps Ability Profile → Career Clusters and skill priorities (Rule Payload). An LLM may write Bahasa Indonesia narrative and action-plan prose strictly inside that payload; on LLM failure, template Insight from the payload still ships.

**Consequences:** Career matrices are versioned config, not prompt folklore; evaluation can assert Rule Payload independently of prose quality.
